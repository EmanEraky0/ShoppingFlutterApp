# used multidependancy:

* dio: For making HTTP requests (good for API calls).
* easy_localization, intl, flutter_localizations: For multi-language support.
* provider, get_it: State management & dependency injection.
* carousel_slider: For image/carousel sliders (like portfolio images).
* flutter_svg: For using SVG icons.
* hive, hive_flutter, shared_preferences: Local storage solutions.
* path_provider: Access local file paths.
* flutter_local_notifications: Push notifications or local alerts.
